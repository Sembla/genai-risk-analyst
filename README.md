# 🚀 GenAI Risk Analyst

Assistente de Inteligência Artificial Generativa para análise de crédito, risco e documentos corporativos.

## 🧠 Sobre o projeto

Este projeto simula um cenário real de uso de IA em empresas como a Serasa Experian, utilizando arquitetura RAG.

## ⚙️ Tecnologias

- Python
- Streamlit
- Scikit-learn

## ▶️ Como executar

pip install -r requirements.txt
streamlit run app.py
